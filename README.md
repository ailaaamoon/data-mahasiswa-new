# Sistem Informasi Data Mahasiswa

Aplikasi CRUD Data Mahasiswa berbasis Node.js dan Express.

## Fitur

- Login Admin
- Tambah Data Mahasiswa
- Edit Data Mahasiswa
- Hapus Data Mahasiswa
- Penyimpanan Data JSON
- Tampilan Responsive

## Teknologi

- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Cara Menjalankan

1. Clone Repository

```bash
git clone https://github.com/ailaaamoon/data-mahasiswa-new.git
```

2. Install Dependency

```bash
npm install
```

3. Jalankan Server

```bash
node app.js
```

4. Buka Browser

```text
http://localhost:3000
```
